SecretPad
==================

SecretPad 是一个基于 [Kuscia](https://github.com/secretflow/kuscia) 的隐私计算的 web 框架，能够方便的使用基于保护隐私的数据智能和机器学习的能力。通过 SecretPad：

* 你可以快速创建节点、注册数据、创建项目、进行合作节点间的授权
* 你可以使用丰富的数据预处理，数据分析、数据建模能力，满足多样化的业务场景。
* 你可以使用模型训练、模型预测的能力。
